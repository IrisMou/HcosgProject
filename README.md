# 捷讯体育

  应用介绍: "捷讯体育"app是一款线上预定乒乓球PK对战、预定乒乓球赛事的应用。

  应用功能: 

    【首页】乒乓球比赛约战、热门场馆、热门竞赛、最新比赛、场馆优选等。
    
    【场馆】精选乒乓球可供PK的场馆，每个场馆下均有不同类型的PK对战。
  
    【PK】关于乒乓球的男子单打、女子单打、男子双打、女子双打等类型PK。
    
    【赛事】精选上海市关于乒乓球的竞赛，用户可通过个人信息直接报名参加。
    
    【我的】我的PK订单、我的报名竞赛、其他相关设置等。
